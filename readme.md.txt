Descripción:
Aplicación web que permite registrar, buscar y listar personas, almacenando la información en el navegador con `localStorage`.

 Funcionalidades:
- Agregar persona con datos personales.
- Buscar por documento, nombre o apellido.
- Listar todas las personas registradas.

 Instrucciones:
1. Clona este repositorio.
2. Abre `index.html` en un navegador web moderno.
